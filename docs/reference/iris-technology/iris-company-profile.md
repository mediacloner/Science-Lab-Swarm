# IRIS Technology Solutions — Company Profile

Source: https://www.iris-eng.com

## Overview

IRIS Technology Solutions is a photonics and AI company specializing in real-time analytical instrumentation for industrial process monitoring and quality control. They develop handheld and in-line spectroscopic analyzers under the Visum brand, combining optical hardware with machine learning software.

## Core Technologies

### NIR Spectroscopy (Near-Infrared)
- Primary technology: analysis of molecular composition via optical radiation interaction with materials
- Wavelength range: 900–1700 nm (SWIR region)
- Applications: moisture, fat, protein, acidity, dry matter, polymeric composition
- Detection limit: ~0.05–0.1% by weight for inline applications
- Results in fractions of a second vs. hours for lab methods
- Supports Process Analytical Technology (PAT) and Quality-by-Design (QbD)

### Raman Spectroscopy
- Vibrational technique based on inelastic scattering of laser light
- High molecular specificity for identification and quantification
- Applied to pharmaceutical process monitoring (e.g., residual solvent tracking during diafiltration)
- Validated against HPLC reference standards

### Hyperspectral Imaging (HSI)
- Combines spectroscopy with spatial imaging
- Enables visual quality control and defect detection
- Material classification and sorting applications

### Machine Vision (DeepSight)
- Automated visual inspection systems
- Deep learning-based pattern recognition
- Defect detection and classification

### AI and Machine Learning
- Chemometric model development (PLS regression, preprocessing optimization)
- Data fusion from multiple spectroscopic sources
- Agent-Based Modelling (ABM)
- Deep learning for spectral analysis
- Visum Master software: automated algorithm selection and model optimization

## Demonstrated Results

### Raman Monitoring of Liposome Diafiltration
- Monitored residual ethanol removal during pharmaceutical liposome manufacturing
- Target: < 0.1% v/v ethanol; training range: 0–10% v/v
- Achieved R² > 0.99, RMSE < 0.35% v/v
- Validated against HPLC reference
- Demonstrates extensibility to other solvent-critical pharmaceutical processes
